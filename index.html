    Parola

    An event based attendance using Beacon Technology

    Get Started →

    Beacon Compatibility

    Parola was design for attendance based so that recording of attendance will be fast as blazing as existing technology. Parola is compatible as long as iBeacon or Eddystone is supported.

    Cross-Platform

    Parola can be implemented for both androi and iOS. Although it is fully implemented in Android devices. The configuration for iOS is complex and therefore discontinued the release build.

    Open Source

    Parola is Open source and the source code will always be free for developers. It is poorly optimized due to time constraint and refactoring is just time consuming.

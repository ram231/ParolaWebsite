          # Why FLUTTER

          Did you know ?

          Flutter was introduced way back 2016 from the project called Sky Engine.

          • Productivity -The Flutter tooling is awesome with code formatting and the readability of the code itself.

          • Familiarity -If you know Java, C++,C# or any Object Oriented Programming Language, you are welcome to Dart's Syntax. WE ALL LOVE Syntactic sugar and that makes us more comfortable.

          • Fast -Flutter can provide 60-120fps(or more). Smooth and Fast Apps is Great Apps.

          • Hot Reload -Tired of Code➡Compile➡Wait➡Error➡Code Cycle? Worry not, Because Flutter has a hot reload feature that instally injects your code to the application without restarting the compiler, so that you can continue to code without compilation interruption.
